<h1>clearText 1.1</h2>

<h2>How to use</h2>
Simply include the <code>jquery.clearText.min.js</code> file and place the following in the head of your document (make sure jQuery is included):

<strong>Minimum setup</strong>
<pre><code>

$(document).ready(function(){
	$('.clear-it').clearText();
});

</code></pre>
